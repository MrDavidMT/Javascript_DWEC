Contents given at DWE Client

Unit 1

Introduction to web programming (JavaScript)

In this unit we will study the integration of html code, the characteristics of JavaScript and its syntax language (variables, types, operators, conditional statements and loops).

Unit 2

JavaScript built-in objects

In this unit we will study the JavaScript native objects(Date, Math, Number, String), Browser Objects(Navigator, Screen, Window, Document, History, Location), and iFrames.

Unit 3

Functions, Arrays and user defined objects

In this unit we will study: JS built-in functions,User defined functions,Arrays,User defined objects,Distributed version control systems: Git & Github.

Unit 4

How to interact with the user?: Events & Forms

In this unit we will study: Events,How to change styles with JS,Forms Validation,Cookies.

Unit 5

Unit 5. Document Object Model (DOM)

In this unit we will study: DOM Objects: properties and methods,Graphics in HTML5: Canvas vs SVG, Graphics Libraries: Google Charts, D3.js, MVC, Storing Objects in HTML5: LocalStorage, SessionStorage.
